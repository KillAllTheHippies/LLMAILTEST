## Getting Started

### System Requirements
- Windows 10/11 or macOS 10.15+
- Python 3.9+
- 4GB RAM minimum

### Installation
1. Download the latest release
2. Extract the package
3. Run the installer:
   ```bash
   python setup.py install
   ```

## Interface Overview

### Main Window Layout
```mermaid
graph LR
    A[Left Panel] --> B[Job Submission]
    C[Middle Panel] --> D[Job Details]
    E[Right Panel] --> F[Job History]
```

### Panel Functions
| Panel | Features |
|-------|----------|
| Left | Scenario selection, Subject/Body input, Rate controls |
| Middle | Job details, Objectives status, Templates |
| Right | Job history, Filtering, Sorting |

## Job Submission

### Basic Submission
1. Select scenario
2. Enter subject and body
3. Set rate limit
4. Click "Submit"

### Status Indicators
| Color | Meaning |
|-------|---------|
| Green | Ready to submit |
| Orange | Rate limited |
| Red | Rate limited with retries |

## Job Management

### Viewing History
- Sort by columns
- Filter by scenario/objectives
- View detailed job information

### Using Templates
1. Select job from history
2. Click "Use as Template"
3. Modify as needed
4. Submit new job

## Advanced Features

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| Ctrl+N | New job |
| Ctrl+T | Use template |
| Ctrl+F | Filter jobs |

### Batch Processing
- Submit multiple jobs at once
- Set global rate limits
- Monitor batch progress

## Troubleshooting

### Common Issues
1. **Connection Errors**: Check network settings
2. **Rate Limiting**: Adjust submission rate
3. **UI Freezes**: Restart application

### Error Codes
| Code | Solution |
|------|----------|
| 400 | Check job parameters |
| 429 | Reduce submission rate |
| 500 | Contact support |

## FAQ

**Q: How do I save my work?**
A: Jobs are automatically saved to `jobs_data.csv`

**Q: Can I export results?**
A: Yes, use the "Export" button in the job history panel

**Q: Where are test reports stored?**
A: Reports are saved in the `test_reports` directory
